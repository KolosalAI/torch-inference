## 📖 Documentation Standards

All documentation in this project follows these standards:

- **Markdown Format**: All docs use standard Markdown
- **Clear Structure**: Hierarchical organization with ToC
- **Code Examples**: Inline code examples where applicable
- **Up-to-date**: Documentation is maintained with code changes
- **Linked**: Cross-references between related documents
